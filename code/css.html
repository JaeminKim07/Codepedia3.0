CSS이란?

CSS는 Cascading Style Sheets 약자로, HTML, XHTML, XML 같은 문서의 스타일를 꾸밀 때 사용하는 스타일 시트 언어입니다.

HTML로 문서의 뼈대를 만들면 CSS는 이 문서의 화장화장을 맡고 있는 셈인데, 글꼴이나, 배경색, 너비와 높이, 위치 등을 지정하거나, 웹 브라우저, 스크린 크기, 장치에 따라서 화면을 다르게 표시될 수 있도록 지정할 수도 있습니다.

CSS는 1996년 12월 W3C웹 문서 표준을 만드는 기관이 도입 했는데 그 전엔 HTML 하나로 문서의 뼈대도 만들고, 꾸밈도 같이 했습니다.

CSS는 문서의 내용과 표현을 분리하여 CSS 파일 하나만 수정하면 스타일에 해당하는 HTML 문서가 한번에 수정하면 스타일에 해당하는 HTML 문서가 한번에 수정되는 엄청난 장점이 있습니다.
